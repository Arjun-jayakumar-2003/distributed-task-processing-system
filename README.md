# Distributed Task Processing System

An asynchronous backend task processing system built with Django, Celery, Redis, and PostgreSQL.

The system supports:

- Background task execution
- Distributed worker processing
- Priority-based task queues
- Automatic retry handling
- Structured logging
- Task lifecycle tracking
- Parallel worker execution

## Tech Stack

- Python
- Django
- Django REST Framework
- Celery
- Redis
- PostgreSQL

## Current Features

- REST API task submission
- Background task processing
- Task status tracking
- Retry handling for failed jobs
- Processing duration tracking
- Structured worker logging
- Multiple worker support
- Priority queue routing
